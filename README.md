# Laboratorio Empresa - Formulario con validaciones

## Description

En este laboratorio se valido los datos de entrada en los inputs del formulario y posterior mostrar cuales son los errores en los inputs.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
